【免费】【开源】【Web】

### 推荐理由

找追剧资源最烦的事不是"没有资源"，而是"不知道哪个资源今天还能用"。
收藏夹里存了一堆网站，打开十个有八个挂了，剩下两个全是广告弹窗。

Awesome Zhuiju Free 解决的就是这个问题：

- 这是一个 GitHub 上公开维护的开源项目，资源数据和检测结果都可查看
- 人工筛选，只收录免费、无广告体验更好的资源
- GitHub Actions 每天早上自动检测每个资源能否访问，结果直接标在 README 和页面上
- 普通用户可以直接浏览，发现失效或推荐新资源也可以通过 GitHub Issue 反馈

### 收录分类

- 在线影视（25 个）
- 影视APP（3 个）
- 网盘资源搜索（4 个）
- 磁力 & BT（9 个）
- 字幕资源（3 个）
- TVBox / 影视仓空壳（2 个）
- TVBox / 影视仓配置地址（16 个）
- 订阅源（1 个）
- 开源项目（10 个）

目前共 73 个资源，持续更新。

### 使用方式

不需要注册、不需要安装。推荐先看 GitHub 仓库，完整资源清单、每日检测状态、数据文件和贡献入口都在那里：

- GitHub 项目主页：https://github.com/laoma2053/awesome-zhuiju-free
- 在线浏览页面：https://zhuiju.me

### 状态说明

每个资源旁边有可用状态标注：🟢 可访问 / 🟡 访问受限 / 🔴 无法访问。
检测由 GitHub Actions 每天北京时间 09:00 左右自动运行，不依赖人工逐个打开测试。

如果你发现某个资源失效，或者有好用的免费无广告资源想推荐，直接在 GitHub 提 Issue 就可以。

说明：Awesome Zhuiju Free 是资源信息导航和可用性状态整理项目，不存储或传播影视文件。第三方资源的内容、访问规则和合规性以对应站点为准；可用性检测结果只代表检测节点当时的网络情况。
