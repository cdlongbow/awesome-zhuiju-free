## 项目简介

SeleneTV 对应的官方项目名是 **Selene**。根据项目 README 说明，它是以 MoonTV v100 版本 / Helios 为后端的客户端，在保留原有体验的基础上，优化了移动端和桌面端的操作体验。

项目基于 Flutter 构建，因此可以覆盖多个平台：

- Android
- iOS
- macOS
- Windows

## 适合谁使用

SeleneTV 更适合下面几类用户：

- 已经了解 MoonTV，希望找一个更顺手客户端的人。
- 想在手机、电脑等不同设备上使用同一套观看体验的人。
- 希望支持播放记录、收藏、断点续播的人。
- 需要 DLNA 投屏，把内容投到电视或大屏设备上的人。

如果你只是想找一个网页打开就能看的影视站，那 SeleneTV 不一定是最简单的选择。它更像是一个完整客户端，适合愿意安装应用、配置和长期使用的人。

## 核心功能亮点

### 1. 多源聚合搜索

SeleneTV 支持多源聚合搜索，可以在多个视频源中查找内容。对用户来说，实际体验就是不用在多个入口之间来回切换，直接搜索片名更省事。

### 2. 播放记录和断点续播

应用会记录播放进度，方便下次继续观看。对于追剧用户来说，这个功能比单纯收藏链接更实用，因为很多时候真正需要的是“接着上次看到的位置继续看”。

### 3. 个人收藏夹

遇到想看的电影、电视剧、动漫或综艺，可以加入收藏。后续再找时不用重新搜索，适合长期追剧或整理片单。

### 4. DLNA 投屏

SeleneTV 支持 DLNA 投屏，可以把内容投到电视或其他大屏设备上观看。手机找片、大屏播放，是比较常见也比较舒服的使用方式。

### 5. 跨平台体验

项目基于 Flutter 开发，并针对移动端和桌面端做了适配。移动端和桌面端使用的播放器技术不同，目的是让各平台都有更合适的播放体验。

## 首次使用流程

首次打开 SeleneTV 时，大致流程是：

1. 启动应用。
2. 应用检查当前登录状态。
3. 如果未登录，会进入登录页面。
4. 登录成功后进入主界面。
5. 在首页查看热门内容、继续观看或个人收藏。
6. 使用搜索功能查找想看的内容。
7. 进入播放器后开始观看，播放进度会自动记录。

不同版本的界面可能会有变化，实际操作以你安装的版本为准。
